by Antonio Garrote <antoniogarrote@gmail.com>

Usage: 
(ns your-namespace
  (:require clj-ml.data-store))

Overview


Functions for storing and retrieving data sets from a persistence store, like a data
base system.
Currently MongoDB is the only store supported.
